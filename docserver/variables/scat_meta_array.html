Workspace Variable scat_meta_array

An Array of scattering meta data (scat_meta).

The elements of the array hold the meta data for single particles used
both for PSD and scattering properties calculations with pnd_fieldSetup,
particle_massesFromMetaDataAndPart_species and scat_data_arrayFromMeta.

Note: This array must contain as many elements as scat_data_array

Usage: Set by the user.

For more details of its content; see also scat_meta.
