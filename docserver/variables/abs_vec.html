Workspace Variable abs_vec

Total absorption vector.

This variable contains the absorption coefficient vector which 
is used in the RTE calculation. It is 
the physical absorption which includes particle absorption 
for all chosen particle types as well as gaseous absorption for
all chosen gaseous species.
The vector is calculated by the agenda opt_prop_part_agenda 
The dimensision of the variable adapts to stokes_dim.

See further the ARTS user guide (AUG). Use the index to find where
this variable is discussed. The variable is listed as a subentry to
"workspace variables".

Usage:      Output of the agenda opt_prop_part_agenda 

Unit:        [Hz, m^2]

Dimensions: [f_grid, stokes_dim]

Group: Matrix
