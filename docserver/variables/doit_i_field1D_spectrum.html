Workspace Variable doit_i_field1D_spectrum

Radiation field for the whole frequency spectrum. 

This variable holds the radiation field. In contrast to 
doit_i_field this variable has an additional freqeuncy 
dimension. This variable is only used for 1D DOIT 
calculations.

Usage: Output of DoitCloudboxFieldPut

Unit: W / (m^2 Hz sr)

Size: [N_f 
       (cloudbox_limits[1] - cloudbox_limits[0]) +1, 
        N_za, N_aa, N_i ]

Group: Tensor4
