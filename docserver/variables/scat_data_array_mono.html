Workspace Variable scat_data_array_mono

Monochromatic single scattering data.

This variable holds the single scattering properties for all 
hydrometeor species. It is calculated from scat_data_array by 
scat_data_array_monoCalc, which interpolates scat_data_array for the 
required frequency.
