Workspace Variable scat_data_array

Raw data of single scattering data.

This variable holds the single scattering properties for all 
hydrometeor species included in a calculation by using the 
methods ParticleTypeAdd or ParticleTypeAddAll. 
For more information refer to ArtsWiki.

Option: See also ScatteringParticleTypeAndMetaRead for reading 
SingleScatteringData and ScatteringMetaData.

The unit of the single scattering properties is m^2.

This may be used in combination with scat_meta_array

Usage: Method ouput.
