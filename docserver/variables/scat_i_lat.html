Workspace Variable scat_i_lat

Intensity field on cloudbox boundary (constant latitude slice).

This variable gives the intensity field from all directions defined 
in scat_aa_grid and scat_za_grid on each grid point on the two 
equal 
latitude levels of the cloudbox boundary. It contains all four 
components of the Stokes vector.

This variable is used as interface between the clear sky and the 
scattering calculations. 

See further the ARTS user guide (AUG). Use the index to find where
this variable is discussed. The variable is listed as a subentry to
"workspace variables".

Usage:      In/Output from/to ScatteringDoit and ScatteringDisort

Unit:        W / (m^2 Hz sr) 

Dimensions: [ f_grid, p_grid, 2, lon_grid, 
              scat_za_grid 
  scat_aa_grid, stokes_dim ]

Group: Tensor7
