Workspace Variable blackbody_radiation

The blackbody radiation for a given temperature.

This variable can be seen as the source term for thermal emission.
Normally, this variable should match the Planck function. The version
of the Planck function taking frequency as input is considered as
default for ARTS. The unit for radiance is then W / [m2 Hz sr].
For frequencies where the Rayleigh-Jeans approximation is valid
(but not recommended) option is to set this variable to be equal
to the physical temperature, resulting in K as unit.

Inside some methods, such as DOIT, the calculation of this source
term can be hard-coded.

Usage:      Set by blackbody_radiation_agenda.

Unit:       See above.

Dimensions: [ f_grid ]
