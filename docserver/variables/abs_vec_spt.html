Workspace Variable abs_vec_spt

Absorption vector for a single particle type.

This variable contains the elements of absorption vector of a 
single particle, given  It is calculated in the agenda 
spt_calc_agenda.

ARTS user guide (AUG) gives the formulas used for computing all 
the elements of absorption vector.

Usage:      Input and Output of the method abs_vec_sptCalc

Unit:        m^2

Dimensions: [N_particletypes,stokes_dim]
