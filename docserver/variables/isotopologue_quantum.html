Workspace Variable isotopologue_quantum

Quantum physical values for the Zeeman effect to work not easily available in other databases.

Example:
This variable should contain total spin [denoted S, with units in angular momentum]
and the molecular Land√© g-factor [denoted g_s, with, to the author, unknown units].

The variable can, e.g., be loaded from a file formated as '@ isotopologue g_s S'
