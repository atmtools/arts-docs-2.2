Agenda spt_calc_agenda

Calculates single particle properties from the amplitude matrix.

This agenda sets up the methods, which should be used to calculate 
the particle properties, i.e. the extinction matrix and the 
absorbtion vector.
 
Normally you  use:
 opt_prop_sptFromMonoData{}
