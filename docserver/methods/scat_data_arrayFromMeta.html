Workspace Method scat_data_arrayFromMeta

Description

This workspace method calculates scattering data and adds it to
scat_data_array using particle meta data in scat_meta_array.
The scattering data is calculated with the T-matrix method.

One set of scattering data is calculated for each particle in
scat_meta_array

Authors: Johan Strandgren, Oliver Lemke
