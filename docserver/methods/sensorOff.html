Workspace Method sensorOff

Description

Sets sensor WSVs to obtain monochromatic pencil beam values.

A 1D antenna pattern is assumed. The variables are set as follows:
   antenna_dim             : 1.
   mblock_za_grid          : Length 1, value 0.
   mblock_aa_grid          : Empty.
   sensor_response*        : As returned by sensor_responseInit.

Authors: Patrick Eriksson
