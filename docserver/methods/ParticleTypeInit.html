Workspace Method ParticleTypeInit

Description

Initializes scat_data_array and pnd_field_raw.

This method initializes variables containing data about the
optical properties of particles (scat_data_array) and about the
particle number distribution (pnd_field_raw)

This method has to be executed before executing e.g.
ParticleTypeAdd.

Authors: Claudia Emde
